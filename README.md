# Simple PHP Api

## Current progress

- Phase (1, GET with Parameters)

  - ?operation=x&numbers=y,z
  - x ... operation (see current operations list)
  - y, z ... simple numbers (1, 10, -20, 80 ...)

## Operations list available

- add (addition / +)
- sub (subtract / -)
- mul (multiply / *)
- div (divide / /)
- mod (modulo / %)
- sqrt (square root / ^2)
